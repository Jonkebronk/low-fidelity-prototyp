# Low-Fidelity Prototype: Digital Information Screen

Digital touchscreen prototype for information sharing in rural Norrbotten, Sweden.

**Course:** I0004E - User Experience Design, Lule√• University of Technology

## Features

- **Home Screen** - Navigation grid + quick info sidebar
- **Events** - Local events with category filters
- **Bus Schedule** - Real-time departures with NEXT badge
- **Weather** - Current conditions + aurora forecast
- **News** - Local news with severity indicators
- **Tourism** - Outdoor activities and attractions
- **Contacts** - Important local phone numbers

## Location

Prototype designed for **Gunnarsbyn**, a village in Boden municipality, Norrbotten.

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Design Principles

- **Accessibility-first** (WCAG 2.1 AA)
- Large text (24pt+ body, 36pt+ headings)
- High contrast (7:1 ratio)
- 48x48px minimum touch targets
- Readable from 2 meters distance

## Personas

- **Erik (27)** - Tech-savvy, wants centralized reliable info
- **Gunnel (78)** - Limited digital skills, needs accessibility
